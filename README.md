## Getting Started <a name = "getting_started"></a>

### Prerequisites

- `python >= 3.9.0`

### Installation

### Format

You can install and run `black`:

```bash
$ pip install black
$ black .
```

Or you can use the
[official VSCode extension](https://marketplace.visualstudio.com/items?itemName=ms-python.python)
that uses `autopep8` by default.
